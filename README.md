## 📌 About

This repository contains the implementation of selected experiments from the **Compiler Design Laboratory**.  
All programs are written in **Python** and demonstrate fundamental compiler construction concepts such as lexical analysis, parsing techniques, grammar transformations, and automata conversion.

---

## 📚 List of Experiments Implemented

1️⃣ Lexical Analyzer  
2️⃣ Regular Expression to NFA Conversion  
3️⃣ NFA to DFA Conversion  
4️⃣ Elimination of Left Recursion  
5️⃣ FIRST and FOLLOW Computation  
6️⃣ Predictive Parsing Table (LL(1))  
7️⃣ Recursive Descent Parsing
